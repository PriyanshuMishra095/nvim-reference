# nvim://reference

A friendly, easy-to-use guide for learning Neovim. It helps beginners understand how to navigate using only their keyboard, built with a clean layout and easy-to-read colors that prevent eye strain.

> **Full Disclosure**: This website and its interactive interface were developed with the assistance of Large Language Models (LLMs) to ensure a state-of-the-art layout, unified styling, and custom mouse physics.

---

## Why nvim://reference?

Traditional Neovim manuals are text-dense, hard to navigate, or lack responsive micro-interactions. **nvim://reference** was built to solve this by providing:
- **Astigmatism-Friendly Typography**: Tailored color contrast matching modern APCA guidelines to reduce eye strain in both light and dark modes.
- **Vim Mode Simulation**: Statusline mechanics and keybinding indicator modes synced dynamically as you browse.
- **Zero Mouse Overhead**: Optimized pathing structure to let developers focus on terminal speed.
- **Interactive Keyboard Controls**: Press `/` to focus search or use custom binds to control UI layout toggles.

---

## Tech Stack

This project is built using a modern frontend stack:
- **Core**: React 18 & TypeScript
- **Bundler & Server**: Vite
- **Styling**: Tailwind CSS & Custom CSS Utilities
- **Icons**: Lucide React
- **Animations**: Framer Motion & Custom JS Repulsion Physics loops

## Getting Started

Follow these steps to run the application locally:

### 1. Clone the Repository
```bash
git clone https://github.com/PriyanshuMishra095/nvim-reference.git
cd nvim-reference
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

The application will run by default at `http://localhost:3000`.
